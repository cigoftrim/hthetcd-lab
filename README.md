# hthetcd-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，主要用于存放各类静态页面资源。

## 项目简介

hthetcd-lab 是一个轻量级的页面归档项目，旨在收集、整理和展示独立的 HTML 文件。这些页面可能来源于不同时期、不同用途的本地测试、原型设计或临时记录。

## 目录说明

仓库根目录下包含以下资源：

- `index.html`：仓库首页入口文件
- `pages/`：存放各个独立 HTML 页面的子目录
- `assets/`：共用的样式、脚本或图片资源
- `archive/`：历史版本或不再活跃的页面归档

## 页面归档说明

每个独立页面通常包含以下内容：

- 一个完整的 HTML 文件
- 必要的 CSS 和 JavaScript 资源（尽量内联或随附在同一目录）
- 简短的说明注释（位于 HTML 文件头部）

页面命名遵循以下规则：

- 以页面功能或主题的英文缩写命名
- 版本号（如有）以 `_v1`、`_v2` 形式附加
- 避免使用特殊字符和空格

## 使用方式

你可以直接通过 GitHub Pages 或其他静态托管服务访问本仓库中的页面。如需在本地查看，请：

1. 克隆本仓库到本地
2. 使用浏览器直接打开对应的 HTML 文件

## 维护说明

- 新页面应放置在 `pages/` 目录下，并确保文件结构清晰
- 不再需要的页面可移入 `archive/` 目录，不建议直接删除
- 共用资源请放置在 `assets/` 中，避免重复引用
- 欢迎提交 Pull Request 完善页面内容或修复问题
